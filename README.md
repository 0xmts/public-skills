# Public Skills

A public collection of skills for vibe coding, design, and product management. In construction.

## Skills

1. [ui-gen](skills/ui-gen/) - Generate composite UI annotation images that pair a polished screen mockup with code-shaped component callouts.

## Install

Ask Codex to use `skill-installer` with this repo and skill path:

```text
Use $skill-installer to install 0xmts/public-skills from path skills/ui-gen.
```

Or use the `npx skills` CLI:

```sh
npx skills add 0xmts/public-skills
```
